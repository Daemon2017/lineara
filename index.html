Справочник слогов

1. Местоимения и личные показатели (Субъекты)

MI : 1-е лицо ед.ч. (Я) / праалтайский *bi (основа *m-)
ME/MI : 1-е лицо мн.ч. (Мы) / восточно-ностратический *-me/*-mi
TI : 2-е лицо ед.ч. (Ты) / индо-алтайский *-ti
TU : 2-е лицо ед.ч. (Ты) / праалтайский и ПИЕ *-tu
SI : 3-е лицо (Он/Тот) / праалтайский *-s- / ПИЕ *-s-
I / E : Указательный маркер (Этот) / ностратический *-i- / ПИЕ *-e-

2. Падежи и локализация (Объект и Пространство)

M* : Аккузатив (Винительный падеж) / ностратический маркер объекта *-m-
NE : Родительный падеж / праалтайский *-ne / ПИЕ *-n
N* : Генитив (Родительный падеж) / праалтайский и ПИЕ маркер *-n-
K* / G* : Датив-Директив (Направление) / ностратический *-k-/*-g-
JA : Локатив (Место) / индо-алтайский суффикс пространства *-ja
TO : Аблатив (Исходный падеж) / индо-алтайский результативный маркер *-to
SA/SE : Партитив / индо-алтайский суффикс относительного признака *-sa/*-se

3. Маркеры числа (Квантификаторы)

R* : Множественность / праалтайский маркер *-ŕ- / ностратический *-r
L* : Плюралис (Собирательное число) / ностратический маркер множественности *-l-
K* / T* : Плюралис именной / праалтайский коллективный суффикс *-k-/*-t-

4. Глагольные показатели (Время, Вид, Залог)

TA : Результативный маркер (Пассив/Аорист) / индо-алтайский *-ta
TU : Имя состояния / номен действия / индо-алтайский результативный маркер *-tu
TI : Презенс/Актив / индо-алтайский глагольный показатель *-ti
B* / P* : Глагол действия / Каузатив / ностратический маркер бытия *-b-/*-p-
NU : Инхоатив (Начало действия) / медиальный глагольный маркер *-nu-
